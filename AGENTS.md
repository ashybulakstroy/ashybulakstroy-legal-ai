# Project settings
- Python virtual environment: `.venv/`
- Для запуска Python всегда используй `.venv/Scripts/python.exe`
- Для pip используй `.venv/Scripts/pip.exe`
- Веди все выводы на русском языке
- для перезагрузки web \ unicorn сервера используй готовые скрипты
- все идее пользователя фиксируй сразу в файле технической спецификации TECHNICAL_SPECIFICATION.md
- веди единый полный план разработки. и внем отмечай что делано.
