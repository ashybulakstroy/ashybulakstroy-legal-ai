# Security Policy / Политика безопасности

## Reporting a Vulnerability / Сообщить об уязвимости

If you discover a security vulnerability, please **do NOT open a public issue**.

Вместо этого напишите нам: **legal-ai-security@example.com** (временно — откройте issue с меткой `security`).

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 0.1  | :white_check_mark: |

## Responsible Disclosure

We kindly ask you to give us reasonable time to fix the issue before disclosing it publicly.
