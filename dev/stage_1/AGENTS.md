# Project settings - dev/stage_1
- Python virtual environment: используем корневой `.venv/` проекта
- Для запуска Python используй `.venv/Scripts/python.exe` (из корня проекта)
- Для pip используй `.venv/Scripts/pip.exe` (из корня проекта)
- PYTHONPATH должен включать `dev/stage_1/` для импорта модулей
- Веди все выводы на русском языке
- для перезагрузки web / unicorn сервера используй готовые скрипты в `dev/stage_1/`
